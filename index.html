Suh dud. We teach you how to use Sahara.

  • Moderation Tools

    Run the command s-kick [Name#0000] [Reason] to kick a user from the guild. Run the command s-ban [Name#0000] [Reason] to ban a user from the guild.

  • Chat Clear

    Run the command s-clear and Sahara will magically delete the fifty recent messages. Messages that are over 14 days old, or two weeks will not be deleted.

  • Music

    Music playback using YouTube, run command s-play [youtube url] and it will play the song, same command to add song to queue, s-skip to skip current song, s-queue to view the queue list.
