Dev Log #001

Written by Jarbone.

Hello children! Welcome to the first week of December, we're rolling out a new features update to Sahara for you! Happy early Christmas!

We have some nifty little features for you and some bug fixes to Sahara, this will all be rolling out to you within the next few weeks time. Firstly, if you would like to view our progress on Sahara, please view our Trello Board for more information (Psst.. Click here dood.) You can also submit new feature suggestions and view the changelog in point-form! You can also submit Bug Reports and stuff on our GitHub page (Which is here, you're welcome.) Also, please join our Discord Guild if you would like priority support! (Top Secret Discord Ninja will beam to your client).

Now onto the Alpha 0.2 Feature Releases!

We have a plentiful amount of stuff for you to toy around with! Be warned that Sahara is currently undergoing an ALPHA phase meaning there will be occasional bugs, please report them on our Github.

Alpha 0.2 New Features

  • New additions to the Music Jukebox, such as queueing skipping songs!
  • Profanity filter, automatic library of swear words!
  • Chat Clearing feature
  • Moderation Tools, Kick, Ban, Soft-Ban, Mute, Soft-Mute

Lots of cool new features for you to play around with, we've also made some dank bug fixes and squashed those disgusting bugs! Catch the Changelog here -

Alpha 0.2 Bug Fixes

  • Running the command "s-play" during a song will stop it and crash bot.
  • Bot will tend to go offline after queue is cleared.

We've also added some new cosmetic changes like a new Discord Bot profile picture, and a Playing Status. Please reinvite the bot to experience changes to the bot.

That's all for now! Catch you later alligator!
