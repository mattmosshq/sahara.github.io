Chat Clearing and Profanity Filter

Sahara works to provide a cool chat filter for those who have sensitive and fragile minds. Our automatic swear filters catch profanity and filters them out so the younger ones can enjoy a safe space to hang out and chill with their other cool friends.. Like Jim!

If the Profanity Filter does not work to perfection, there is a way to remove unnessecary spam or other swear words by running a certain command. Please note, messages that are older then 2 weeks or 14 days will not be removed. Also, currently Sahara will delete a message count of fifty.

Command Name Description Command
Chat Clear Bulk deletes unwanted spicy meme messages by a message count of fifty. s-clear

Music Playback

Sahara supports Music Playback in voice channels. This will be done by fetching YouTube videos and playing it on voice channels. Soon, Sahara hopes to offer Spotify streaming services. Sahara can play songs, add new songs to the queue, skip songs, and view the queue.

Command Name Description Command
Play Music Play music and add music to the queue. s-play [url]
Skip Music Skip the current playing song, if there is more then one person in the voice channel, a vote will be in place and majority will rule the vote. s-skip
View Queue View the current queue list and the current song that is playing. s-queue

Moderation Tools

Sahara supports certain moderation tools for Moderators and Guild Administrators. These features include Kick Members, Ban Members, Soft-Ban Members, and Mute / Soft Mute players from voice channels and text channels. More Moderation features will be arriving soon! Please check our Dev-Logs for more!

Command Name Description Command
Kick Members Kick Members from the selected Guild. s-kick [Name#0000]
Ban Members Permanent Ban Members from the selected Guild. s-ban [Name#0000]
Soft Ban Members Temporarily Ban Members from the selected Guild. s-ban [Name#0000] [Time: DD:HH:MM]
Mute Members Permanent Mute Members from the selected Guild. s-mute [Name#0000]
Soft Mute Members Temporarily Mute Members from the selected Guild. s-mute [Name#0000] [Time: DD:HH:MM]
